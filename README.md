# HackYard
**Gardening playing simple**

![](images/screenshot1.png)

## Start Development
```
npm install
```

## Run backend server
```
cd server
nodemon server 
```

## Run frontend
```
ionic serve
```

## Seed database
```
mongoimport --db mongodb --collection plantRecords --file plantDB.json --jsonArray
```
